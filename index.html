PayPlus x ToyyibPay

Jana bil untuk NFC tag dan QR terus dari browser.

Versi ini tak pakai backend. Anda cuma masukkan jumlah, sistem akan terus cipta bil ToyyibPay dan jana QR code pada halaman yang sama.

Mod frontend-only ini dedahkan user secret key dalam source code. Kalau nak lebih selamat, backend tetap diperlukan.
Bil kini dijana dengan saluran FPX + kad. NFC/payWave sebenar tak boleh diakses terus oleh laman web biasa, jadi telefon masih akan bayar melalui halaman hos ToyyibPay selepas pautan dibuka.
NFC tag hanya boleh ditulis pada pelayar yang menyokong Web NFC dalam secure context seperti https:// atau localhost. Ini bukan terminal payWave sebenar.

Jana bil baharu

Field lain telah dibuang. Nama, email, telefon, rujukan dalaman dan deskripsi kini tidak perlu diisi oleh pengguna. Halaman bil ToyyibPay akan buka dengan pilihan pembayaran bank dan kad.
